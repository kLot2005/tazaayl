# 🗺 Дорожная карта фронтенда Tazaayl

## ✅ Завершено (Done)
- [x] **Инициализация системы**: Next.js 16+, Framer Motion, Tailwind (кастомный glassmorphism).
- [x] **Авторизация**: Защищенный вход/JWT поток с навигацией по ролям.
- [x] **Главный дашборд**: Глобальные виджеты (машины, зоны, маршруты).
- [x] **Интерактивная карта (2D)**: Интеграция Mapbox GL, операционный 2D-режим.
- [x] **Живой трекинг**: Отображение автопарка в реальном времени через Socket.io.
- [x] **Оцифровка зон**: Инструменты рисования полигонов GeoJSON прямо в браузере.
- [x] **Управление автопарком**: CRUD-интерфейс для мусоровозов и спецтехники.
- [x] **Управление зонами**: Отдельный дашборд для управления именами и цветами полигонов.
- [x] **Режим мониторинга**: Полноэкранный режим для ситуационных центров.

## 🚀 Будущие фазы (В планах)

### ФАЗА 1: Экосистема водителя (Mobile-First) 📱
*Цель: Мобильное PWA для полевых сотрудников.*
- **Авторизация водителя**: Упрощенный вход (по ID или номеру телефона).
- **Панель управления**: Большие кнопки **"Начать смену"** / **"Завершить смену"**.
- **GPS Движок**: Фоновое отслеживание местоположения с оптимизацией батареи.
- **Отчетность**: Быстрые кнопки "Контейнер полон" / "Проезд заблокирован".
- **Навигация**: Минималистичный вид карты с границами назначенного сектора.

### ФАЗА 2: Аналитика и отчеты (Insights) 📊
*Цель: Превращение перемещений в данные.*
- **Тепловые карты**: Визуализация плотности и частоты сбора мусора.
- **Верификация сервиса**: Дашборд "Подтверждение посещения" (какие улицы пропущены?).
- **Метрики эффективности**: Средняя скорость, время простоя и оценка расхода топлива.
- **Система экспорта**: PDF/Excel отчеты для городской администрации.

### ФАЗА 3: Оптимизация маршрутов 🧠
*Цель: Умная логистика.*
- **Расчет пути**: Генерация оптимального маршрута между точками сбора.
- **Назначение зон**: Интерфейс Drag-and-drop для закрепления машин за секторами.
- **Алерты об отклонении**: Мгновенные уведомления, если машина покидает назначенный сектор.

### ФАЗА 4: Административное управление ⚙️
- **Логи аудита**: История всех изменений зон и машин.
- **Управление пользователями**: Создание и настройка аккаунтов операторов.
- **Настройки системы**: Ключи API, токены Mapbox и конфигурация Telegram-бота.

---

### 🛠 Стек технологий
- **Frontend**: Next.js, Framer Motion, Lucide, Socket.io-client.
- **GIS**: Mapbox GL JS, Mapbox-Draw.
- **Mobile**: Service Workers (PWA), Geolocation API.
- **State**: React Context / Hooks.
